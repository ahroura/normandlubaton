Database Management System

STACKS USED
• Visual Basic • MySQL

This is a DBMS web application created together with my partner as a final project for a Database subject. The premise is that IKEA has commisioned me to create a DBMS for their stocks. This DBMS includes 2 seperate interfaces; one for the admin and the other for the customer. The admin is able to view all the data in the database and perform basic CRUD(Create, Read, Update, Delete) function while the customer has the ability to customize their profile, create a new order and track their current orders.

Discord Bot

STACKS USED
• Node.js • Discord.js

A bot created as a final project for a UX subject. The premise of the project is to create a bot specifically in Discord that has the functionality that people uses in order to lessen the confusion caused by the presence of multiple bots in a Discord server. Multiple interviews has been conducted as a method of need finding and recommended features by interviewees has been implemented into the bot using mainly Javascript. The bot contains multiple commands such as
• Music commands that uses Youtube API and ffmpeg in order to play music through Discord's voice call.
• Utility commands that uses Discord.js which eases direct communication with Discord API to manage a server.
• Miscellaneous commands that utilizes rand() function in Node.js to create simple games.
